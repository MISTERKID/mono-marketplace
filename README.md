## ğŸš€ Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Quick Start

```bash
# Clone the repository
git clone https://github.com/MISTERKID/mono-marketplace.git

# Navigate to project directory
cd mono-marketplace

# Install dependencies
npm install
# or
yarn install

# Start development server
npm run dev
# or
yarn dev
```
